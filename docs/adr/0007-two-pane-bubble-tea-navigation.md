# ADR 0007: Two-Pane Bubble Tea Navigation

**Status:** Accepted

The Go MVP will keep the product idea of a two-pane TUI: left pane for context/navigation and right pane for the active MR detail or diff workflow. The implementation will be Bubble Tea-native with focusable panes, keyboard navigation, and mouse selection/clicks rather than a React-style Navigation Stack or context-based screen components.
